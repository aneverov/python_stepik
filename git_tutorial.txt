1. На сайте github создаю проект.
2. создаю файлы на компьютере(проект)
3. перехожу в эту папку(проекта)
4. git status - показывает есть ли изменения на компе и в гитхабе
5. git add имя_файла - добавляет файл (сравнивается с гитхабом и готов в него загрузиться)
6. git commit -m "Указываем внесенные изменения"
7. git push - закидывается файл(-ы)

8. git pull - скачивает недостающие файлы или 
9. git checkout -- <file name> возвращает удаленный файл из гитхаба на комп
